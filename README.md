# Task2-html# MarketYard – Crop Price Dashboard

MarketYard is a front-end data visualization dashboard that helps farmers, traders, and
agricultural stakeholders explore crop prices across regions and time. It runs completely
on the browser using HTML, CSS, JavaScript, and Chart.js, with no backend required.

The dashboard displays historical price trends, comparison charts, summary statistics,
and recent market movements, making crop market insights easier to interpret.

---

## Features

### ✔ Core Functional Requirements
- Browse and search price data by **commodity**, **region**, and **date range**
- View **interactive line charts** of historical prices
- Compare commodities side-by-side using an additional chart
- Switch between **daily, weekly, or monthly** data aggregation
- Export current chart data as **CSV**
- Automatic detection of spikes and drops with signal tags
- Responsive design suitable for laptop and mobile

### ✔ Optional Enhancements (Supported / Extendable)
- Regional map integration using a mock map API
- Price spike/dip threshold alerts
- Moving average trend indicators (extendable)
- CSV import so users can load their own price dataset

---

## Technology Used
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Chart.js (line charts & datasets)**
- No backend required (all data generated in-browser)
